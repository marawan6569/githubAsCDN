## Spacing

The framework provides classes for spacing. Here are some examples:

| Class   | Description                |
|---------|----------------------------|
| .mt-1   | Margin top of 0.25rem      |
| .mr-2   | Margin right of 0.5rem     |
| .mb-3   | Margin bottom of 1rem      |
| .ml-4   | Margin left of 1.5rem      |
